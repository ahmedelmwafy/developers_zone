# Developers Zone 🚀

**Where Code Meets Community.**

Developers Zone is a premium, high-performance social networking platform designed specifically for developers to connect, share insights, and collaborate. Built with Flutter and Firebase, it offers a seamless bilingual experience (English & Arabic) with a focus on rich aesthetics and real-time interactions.

---

## ✨ Core Features

- **Bilingual Support**: Full localization for English and Arabic (RTL support).
- **Real-time Connectivity**: Instant messaging and social interactions powered by Firestore.
- **Advanced Networking**: Detailed developer profiles with verification badges.
- **Global Social Feed**: Share thoughts, tips, and code snippets with image support.
- **Moderation & Safety**: Multi-layer admin controls and user-side blocking/reporting system.
- **Dynamic Monetization**: Integrated Ad Management system for Splash and Home banners.

---

## 📱 App Screens & UI Description

### 🌟 1. Splash & Authentication
- **Splash Screen**: Feature-rich animated entry with particle effects and dynamic splash ads.
- **Smart Auth**: Clean, modern interface for Email/Password login and registration. Includes social sign-in integration.
- **Waiting Approval**: A secure gatekeeping screen for new users awaiting administrative review.

### 🏠 2. Home & Global Feed
- **Home Hub**: Central navigation using a sleek bottom bar with micro-animations.
- **Global Feed**: A scrollable stream of developer posts. Features like/unlike, commenting, and image galleries.
- **Create Post**: Full-screen intuitive editor with multi-image upload support via ImgBB.

### 🔍 3. Explore & Search
- **Developer Discovery**: Real-time search engine to find developers by name, job position, or city.
- **Collaboration Finder**: Specifically designed to help users find their next project partner.

### 💬 4. Real-time Chat
- **Chat Network**: List view showing active conversations with unread message indicators.
- **Advanced Controls**: Swipe actions to archive, delete, or mute notifications for specific chats.
- **Direct Messaging**: Fluid, real-time message exchange with "last seen" and "online" status.

### 👤 5. Developer Profile
- **Comprehensive View**: Stats (Followers/Following), Bio, Job Position, and Social Links (GitHub, LinkedIn, Portfolio).
- **Verification System**: Visual progress tracker for profile completeness to earn the verified badge.
- **Social Actions**: Follow/Unfollow, Message, and Block/Report capabilities.

### ⚙️ 6. Settings & Security
- **Account Management**: Centralized hub for profile editing and security updates.
- **Security Suite**: Modal-based password change and sensitive action confirmations.
- **Preferences**: Real-time language switching and granular notification settings.

### 🛡️ 7. Admin Dashboard
- **User Moderation**: Holistic view of the community with "Approve", "Verify", "Ban", and "Make Admin" actions.
- **Live Stats**: Real-time counters for Total, Approved, and Banned users.
- **Ad Management**: Dedicated interface to upload and manage localized ad banners.

---

## 🛠️ Technology Stack

- **Frontend**: Flutter (Dart)
- **Backend**: Firebase (Auth, Firestore, Cloud Messaging)
- **Storage/CDN**: ImgBB API for optimized image hosting
- **Localization**: Custom AppLocalization with JSON assets
- **Aesthetics**: Custom-designed dark theme with glassmorphism and vibrant gradients

## 🛠️ Getting Started

1. **Clone the repository**
2. **Install dependencies**: `flutter pub get`
3. **Setup Firebase**: Add your `google-services.json` (Android) and `GoogleService-Info.plist` (iOS).
4. **Run the app**: `flutter run`
